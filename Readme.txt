Установка:
Шаг 1: Распакуйте архив в какую-нибудь папку на компьютере (оба файла)
Шаг 2: Запустите setup.exe
Шаг 3: Следуйте указаниям установщика. 
Шаг 4: Вы восхитительны!
Удаление:
Шаг 1: Запустите файл setup.exe
Шаг 2: Выберите пункт "Удалить "Twenty Points to Win"
Шаг 3: Следуйте указаниям установщика. 
Шаг 4: Вы восхитительны!
